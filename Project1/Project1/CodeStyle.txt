CodeStyle for José Mª Contreras on project "Bomberman"

camelCase (primera minuscula, demes majuscules) 
PascalCase (la pimera amb majuscula)
snake_case(separa parules amb "_")
Hungrian Notation (indicar amb una lletra el tipos de variable que es ej: bool bpepito)

FileName:
- Will use PascalCase
- Example: String.h
Vaiables:
- Will use snake_case
- Example: size_memory
Class Names:
- Will use PascalCase
- Example: class String()
Method Names:
- Will use camelCase
- int clear()
